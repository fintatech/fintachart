# Introduction

FintaChart is a multi-platform financial charting component for web applications. It provides a complete, production-ready charting solution for trading platforms, financial dashboards, and any HTML-based project that needs interactive market data visualization.

## Key Features

- **16+ chart types** -- OHLC, Candle, Heikin Ashi, Renko, Kagi, Point & Figure, and more
- **100+ technical indicators** -- Moving averages, oscillators, volatility, volume, and custom indicators
- **Drawing tools** -- Trend lines, channels, rays, and freehand drawing
- **Fibonacci tools** -- Retracements, extensions, fans, arcs, and time zones
- **Trend analysis** -- Linear regression, trend channels, and speed resistance
- **Pattern recognition** -- Built-in support for common chart patterns
- **10 themes** -- Light, dark, and specialty themes with full customization
- **Multi-chart layouts** -- Display multiple synchronized charts side by side
- **Calendar events** -- Overlay economic events and earnings directly on charts
- **Localization** -- Ship with English and Ukrainian; extend with custom locales
- **Touch support** -- Optimized gesture handling for mobile and tablet devices
- **State management** -- Save, restore, and template entire chart configurations

## Browser Compatibility

| Browser | Minimum Version |
|---------|----------------|
| Chrome | 29+ |
| Safari | 9+ |
| Firefox | 28+ |
| Opera | 17+ |
| Edge | 17+ |
| Android Browser | 5+ |

FintaChart uses HTML5 Canvas for rendering and does not require Flash or any other plugin.
